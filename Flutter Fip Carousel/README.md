## Code Example:
```dart

  // List of widgets to display (infinite :D)
  items: [ImageOne(), ImageTwo()],
  // Callback function triggered on item change
  onChange: () {},
  // Width of the FlipCarousel widget
  width: 250,
  // Height of the FlipCarousel widget
  height: 400,
);
```
